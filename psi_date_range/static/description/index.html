SHORT OWERVIEW

The module helps to generate and delete sequence_date_range for selected sequences in monthly and current year ranges with a single click of a button.

SCREENSHOTS

DATE RANGE GENERATOR

SETTIINGS --> TECHNICAL --> SEQUENCES

Go into options, then technical and sequences and then select one.

When Use subsequences per date_range is checked, an additional field Generate next number by appears

From the select list we can choose to generate a monthly range, or the current year, then click the GENERATE button. Additionally, the REMOVE ALL button allows us to quickly remove all sequence_date_range for this sequence.
